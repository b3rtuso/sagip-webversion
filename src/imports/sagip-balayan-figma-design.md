Create a modern dashboard-style web application UI in Figma for a system called:

"Sagip Balayan: Disaster Incident Reporting and Response Management System"

The system will be used by the MDRRMO of Balayan, Batangas to manage disaster reports, emergency response dispatching, personnel, and equipment.

Design Style:

* Clean government-style interface
* Blue and white color scheme
* Dashboard layout
* Left sidebar navigation
* Top header bar
* Cards, charts, and data tables
* Professional and minimal design

Create the following pages:

1. Login Page

* System logo
* Title: Sagip Balayan
* Username field
* Password field
* Login button
* Clean centered layout

2. Dashboard Page

* Sidebar navigation with icons:
  Dashboard, Incidents, Dispatch, Personnel, Inventory, Analytics, Reports, Settings
* Top header with system title and user profile
* Statistic cards:
  Total Incidents
  Active Incidents
  Resolved Incidents
  Average Response Time
* Line graph showing monthly incidents
* Pie chart showing incident types (Fire, Flood, Accident, Landslide)
* Recent incidents table
* Map or hotspot area panel for high-risk barangays

3. Incident Reporting Page

* Incident report form with fields:
  Reporter Name
  Contact Number
  Barangay dropdown
  Incident Description text area
  Upload photo
* AI prediction fields:
  Predicted Incident Type
  Priority Level (High, Medium, Low)
* Submit button
* Incident table showing:
  ID, Barangay, Incident Type, Priority, Status, Date

4. Dispatch Management Page

* Incident queue list
* Assign response team dropdown
* Assign vehicle/equipment
* Dispatch button
* Incident status tracking:
  Pending, Dispatched, On Scene, Resolved

5. Personnel Management Page

* Table listing responders
* Columns: Name, Role, Team, Status
* Add personnel button

6. Equipment & Inventory Page

* Equipment list:
  Fire trucks, rescue kits, medical supplies
* Status indicators: Available, In Use, Maintenance
* Relief goods inventory table

7. Analytics Page

* Graphs showing incident trends
* Response time analytics
* Incident distribution by barangay
* Dashboard charts

8. Reports Page

* Generate report panel
* Filters: Date range, Incident type, Barangay
* Export report button

Design the layout as a responsive desktop web application with consistent spacing, icons, and modern UI components.

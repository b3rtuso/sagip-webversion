Create a complete mobile application UI design in Figma for a disaster management system called:

"Sagip Balayan: Disaster Incident Reporting and Response Management System"

The app will be used by responders and MDRRMO staff in the field to report incidents, monitor emergencies, and manage disaster response operations in Balayan, Batangas.

Design Requirements:

* Mobile-first interface
* Android-style layout
* Clean and professional government application design
* Blue, white, and light gray color palette
* Simple and easy-to-use interface for emergency situations
* Use icons and cards for quick information
* Frame size: Mobile (390px × 844px)

Create the following mobile screens:

1. Splash Screen

* App logo
* App name: Sagip Balayan
* Tagline: Disaster Response Management System

2. Login Screen

* Username field
* Password field
* Login button
* Clean centered layout
* Optional “Remember Me” checkbox

3. Home Dashboard

* Greeting header (e.g., "Welcome, Responder")
* Summary cards showing:

  * Active Incidents
  * Pending Dispatch
  * Completed Responses
* Quick action buttons:

  * Report Incident
  * View Incidents
  * Dispatch Updates
* Recent incidents list

4. Report Incident Screen

* Form fields:

  * Reporter Name
  * Contact Number
  * Barangay dropdown
  * Incident Description text area
  * Upload photo or take photo
* AI-generated fields:

  * Predicted Incident Type
  * Priority Level (High, Medium, Low)
* Submit Incident button

5. Active Incidents Screen

* Scrollable list of incidents
* Each card shows:

  * Incident type
  * Barangay location
  * Priority indicator
  * Status (Pending, Dispatched, On Scene, Resolved)
* Tap card to view details

6. Incident Details Screen

* Full incident information
* Location and description
* Assigned team and equipment
* Buttons for responders:

  * Mark "On Scene"
  * Mark "Resolved"

7. Dispatch Updates Screen

* List of assigned incidents
* Update status
* Record arrival time
* Add response notes

8. Analytics Screen

* Simple mobile charts
* Total incidents this month
* Most common disaster type
* Average response time

9. Profile Screen

* User profile information
* Role (Responder / Dispatcher)
* Logout button

Design the layout using modern mobile UI components such as cards, rounded buttons, icon navigation, and bottom navigation tabs.

Ensure the interface is simple, readable, and optimized for emergency field use.
